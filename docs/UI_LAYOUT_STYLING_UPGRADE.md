# Layout & Styling Upgrade

The trading terminal UI has been upgraded to a modular, professional-grade layout with dynamic visual feedback.

## Snap-to-Grid Workspace

The fixed static layout has been entirely replaced with **FlexLayout**. You can now:
- **Tear off tabs** into floating windows (perfect for multi-monitor setups).
- **Drag and drop** panels to create custom grid layouts.
- **Resize panels** dynamically anywhere on the screen.

The layout manager handles the Watchlist, Chart Grid, Trading Panel, and Dock as first-class dockable components.

## Trust Micro-Interactions

The order book and trade history have been updated with **institutional-grade micro-interactions**:
- **Order Book Flashing**: The top of the order book (best bid/ask) will flash green or red whenever the spread or top-of-book size updates.
- **Trade History Flashing**: Newly filled trades will briefly flash green (for buys) or red (for sells) in the blotter, providing instantaneous visual feedback of execution.

> [!TIP]
> Try dragging the "Watchlist" tab and dropping it onto the right side of the screen, or drag a chart outside the browser window entirely to pop it out!
